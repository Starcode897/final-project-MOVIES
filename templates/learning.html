Machine Learning


We designed a multivariable linear regression model in order to estimate monthly cost. Below is a residual plot that demonstrates provides insight on the model's accuracy. This model's R2 value was .98.


The Process:


To prepare the data for the model, we studied the correlation coefficients among the variables. We eliminated variables that would cause multicollinearity, and chose variables that would likely have an effect on monthly price (we used the correlation coefficients as a guide). To address categorical variables, we applied one-hot encoding. The data was then split into training and testing data for the machine learning process. As each model provided a different R2 and MSE value, we compared the values in order to choose our best model.
